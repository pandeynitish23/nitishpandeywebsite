- **LinkedIn** → Opens your LinkedIn profile for direct messaging

## 🌐 Deploy to GitHub Pages

1. Create a new GitHub repository (e.g., `nitishkumarpandey-cv`)
2. Upload all files in this folder to the repo root
3. Go to **Settings → Pages → Source → Branch: main → Folder: / (root)**
4. Click **Save** – your site will be live in ~1 minute!

> **Tip:** Update the LinkedIn URL in `index.html` with your actual LinkedIn profile slug.
